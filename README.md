# relatorio-sac-bi
# 📊 Dashboard de Relatório de Saques

Este projeto apresenta um dashboard desenvolvido em Power BI para análise de atendimentos relacionados a saques, permitindo a visualização de indicadores operacionais e de performance.

---

## 🎯 Objetivo

Centralizar e analisar dados de atendimentos, permitindo uma visão clara sobre volume de chamados, eficiência operacional e qualidade do atendimento.

---

## 📈 Principais indicadores

O dashboard permite visualizar:

- 👥 Total de atendimentos por agente  
- 📊 Total geral de atendimentos  
- ⏱️ Tempo médio de atendimento (em minutos)  
- 🔁 Tempo médio de retorno (em dias)  
- 📅 Média diária de chamados  
- ❌ Percentual de chamados cancelados  
- 📆 Total de atendimentos por ano e mês  
- 🧾 Atendimentos por tipo de problema  
- 👤 Atendimentos por nome de atendimento  

---

## 🛠️ Ferramentas utilizadas

- Power BI  
- Modelagem de dados  
- Análise de KPIs operacionais  

---

## 📁 Estrutura do projeto

- `/relatorio` → Arquivo do Power BI (.pbix)  
- `/imagens` → Prints e imagens do dashboard
